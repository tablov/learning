Базовые элементы JS

Функция без аргументов, ничего не возвращает

            function f1(){
                alert('It works!');
            }
            // когда мы ее вызываем, выскакивает модальное окно 'It works!'

Функция с одним аргументом, ничего не возвращает

            function f2(x){
                alert(x * 2);
            }
            // когда мы ее вызываем, выскакивает модальное окно c удвоенным значением аргумента

Функция с двумя аргументами, ничего не возвращает

            function f3(arg1, arg2){
                alert(arg1 + arg2);
            }
            // когда мы ее вызываем, выскакивает модальное окно c суммой аргументов

Функция с двумя аргументами

            function f4(arg1, arg2){
                return arg1 * arg2;
            }
            // когда мы ее вызываем, она возвращает произведение аргументов

Функция с двумя аргументами

            function f5(arg1, arg2){
                alert(arg1 * arg2);
                return 10;
            }
            // когда мы ее вызываем, выскакивает модальное окно c произведением
            // аргументов, а возвращает она всегда 10
